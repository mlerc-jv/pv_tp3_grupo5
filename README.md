## Descripcion TP N°3

El TP N°3 consiste en la evolucion de la plataforma  "Gestión de Proyectos Educativos" desarrollada en el TP N°1 (con HTML y CSS), migrandola a React utilizando Vite para que sea una aplicacion dinamica e interactiva para el usuario.

Esta aplicacion permite visualizar, buscar, agregar y eliminar proyectos en tiempo real usando componentes funcionales, hooks como useState y renderizado dinamico con .map()

Por ultimo, la estructura del trabajo fue organizado en componentes reutilizables, un archivo CSS para cada componente y un modulo de servicios encargado de administrar la logica de la aplicacion.
